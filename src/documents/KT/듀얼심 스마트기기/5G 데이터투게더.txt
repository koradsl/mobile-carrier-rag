** 데이터 제공 안내 **

- 공유
    - 3GB는 스마트폰 데이터와 합산하여 스마트기기(패드/노트북) 및 스마트폰에서 사용됩니다.
    - 3GB 는 5G/LTE 스마트폰 요금제의 이월 및 기본제공 데이터보다 우선 차감됩니다.
    - 공유 회선 해지 시 3GB 데이터는 일할 계산되어 차감되며, 초과 시 5G/LTE스마트폰 요금제의 기본 제공 데이터에서 차감됩니다.
    - 5G/LTE 스마트폰 요금제별 데이터 공유 범위(예시)
    <표>
    | 5G / LTE 스마트폰 요금제 | 공유 범위 | 공유 데이터 소진 후 요금 |
    | 초이스 프리미엄 | 3GB + 100GB(스마트기기 / 테더링 데이터 한도) | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    | 초이스 스페셜 | 3GB + 70GB(스마트기기 / 테더링 데이터 한도) | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    | 스페셜/ Y 스페셜 | 3GB + 50GB(스마트기기 / 테더링 데이터 한도) | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |

    | 5G 슬림 14GB | 3GB + 14GB(기본 제공량) | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    | 데이터ON 프리미엄 | 3GB + 50GB | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    | 데이터 일 제공량 있는 요금제 | 3GB + 기본 제공량 + 일제공량 | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    | 데이터 일 제공량 없는 요금제 / 안심 차단형 요금제 | 3GB + 기본 제공량 | 미발생(최대 200Kbps 속도로 지속 이용), 안심 차단형 요금제는 공유 데이터 소진 후 차단 |
    </표>

- 공통
    - 3GB데이터는 이월 및 당겨쓰기가 되지 않습니다.
    - 단독 → 공유 혹은 공유 → 단독으로 변경 시, 3GB데이터는 일할 제공 됩니다.
    - 본 요금제에 가입된 단말기에 한하여, olleh WiFi 서비스가 기본 제공됩니다.


** 가입 및 유의사항 **

- 5G/LTE 패드, 노트북 등 스마트기기 단말만 가입 가능합니다.

- 5G/LTE 스마트폰과 동일 명의인 경우 데이터 공유 및 할인이 가능합니다.

- 문자 수신이 불가한 단말기는 ‘데이터 전용 단말 통보 서비스’ 별도 가입 후 스마트폰에서 데이터사용량 문자를 받으실 수 있습니다.

- 월 중 가입 또는 변경하는 경우, 변경 시점 부터 월정액 및 기본 제공 데이터가 일자별 계산하여 제공 됩니다.
