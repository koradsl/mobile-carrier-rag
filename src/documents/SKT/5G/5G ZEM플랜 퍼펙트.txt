** 이용 요금 및 혜택 **

- 1. 가입 조건

    - 만 12세 이하 개인명의 고객님에 한하여 1인 1회선 가입 가능

    - 5G/LTE 스마트폰으로 가입 가능
        - 단 5G 스마트폰을 사용하실 경우에만 5G 데이터를 이용하실 수 있습니다.

- 2. 기본제공 혜택

<표>
| 요금제 | 월정액(부가세 포함, *선택약정 반영 금액) | 음성/문자 | 데이터 |
| 5G ZEM플랜 퍼펙트 | 36,000원(26,980원) | 집전화/이동전화 무제한+영상/부가300분+문자무제한 | 4GB 이후 최대 1Mbps 속도로 계속 사용 |
| 5G ZEM플랜 베스트 | 26,000원(19,455원) | 집전화/이동전화 무제한+영상/부가100분+문자무제한 | 2GB 이후 최대 400kbps 속도로 계속 사용 |
</표>

- 3. 데이터 혜택 상세

    - 데이터 함께쓰기/테더링 서비스의 이용 한도는 별도 제공되지 않고, 해당 서비스 이용 시 요금제의 기본 제공 데이터가 동시에 차감

    - 리필한 데이터는 데이터 함께쓰기, 데이터를 사용한 음성/영상통화(m-VoIP) 등에 사용 불가

    - T 가족모아데이터 서비스를 이용 불가

    - ZEM app 발생 데이터 비과금 적용

- 4. 부모(법정대리인)의 데이터 제한 설정

    - 자녀가 요금제의 기본 제공 데이터를 다 쓰면 데이터를 추가 사용 할 수 없도록 제한하는 기능

    - 법정대리인으로 등록된 가입자의 T world에서 제한 설정 및 상태 조회 가능 (App: 나의 데이터/통화 > 자녀 실시간 잔여량, PC: myT > 데이터/통화 관리 > 나의 잔여 데이터/통화)

    - 데이터 제한 설정은 일 1회만 가능 (요금제 가입 당일은 일 2회)

- 5. 기타 유의 사항

    - 부가통화 제공량을 모두 쓰면 부가통화 및 영상통화를 걸 수 없음

    - Fair Use Policy에 따라 문자메시지 이용이 제한되어 건별 과금 시 적용 요금 : SMS 22원/건, 텍스트형 MMS 33원/건, 멀티미디어형 MMS 110원/건

    - 기본제공데이터 소진 후 적용되는 속도제어 해제 불가

    - 음성/문자 이용 시 정보료가 발생하는 서비스(060음성, 문자투표 등의 유료 문자)를 이용 할 시 정보료 요금 발생

    - 선불 및 후불 충전을 통한 통화료 차감 불가

    - 만 19세 미만 청소년 고객(어린이 포함) 고객은 T 끼리 데이터 선물하기 서비스를 이용할 수 없으며, 선물 받기만 가능. 단 아래의, 경우에서만 예외적으로 선물하기가 가능.

    - 만 19세 미만 청소년 고객(어린이 포함)은 데이터 선물하기 가능 요금제를 사용하고 있고, 가족간 추가로 선물횟수를 주는 결합상품에 가입되어 있는 경우 법정대리인 동의 하에 결합가족 회선에게만 선물하기 가능

    - 아래 경우에는 이 요금제의 이용이 제한될 수 있습니다.

        - 문자(SMS/MMS)를 광고성 스팸 문자 등 상업적 목적으로 발송하거나 물리적 장치 또는 자동 발송 프로그램을 이용하여 발송하는 경우

        - 월 3,000개 이상의 전화번호에 문자를 발송하는 경우 광고/상업적 목적의 이용으로 간주함
            - 요금제 이용이 제한되는 경우, 이 요금제에 가입하신 휴대폰 번호로 문자로 안내드립니다. 이후 문자 요금은 사용하신 만큼 청구됩니다.(단, 광고/상업적 목적으로 이용하신 경우 요금제가 <표준요금제>로 자동 변경됩니다.)
